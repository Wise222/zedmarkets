# ZedMarkets
Zambia's trusted marketplace. Buy smart. Sell confidently.

## What this is
ZedMarkets connects verified Zambian sellers with buyers who are tired of getting scammed on WhatsApp and Facebook. Every seller is checked before they can list anything.

## Built with
React, CSS, JavaScript

## Folder structure
public        Static files
src           All source code
src/components    Reusable pieces like navbar and product cards
src/pages         One file per page
src/assets        Images and icons
src/styles        CSS files
src/data          Sample product and seller data
docs          Project notes
